# Amazon Athena

Amazon Athena is an interactive analytics service that makes it simple to analyze data directly in Amazon S3 using standard SQL. Athena is serverless, so there is no infrastructure to manage, and you pay only for the queries that you run.

## APIs

- **Amazon Athena API** - The Amazon Athena API enables programmatic access to run and manage SQL queries against data stored in Amazon S3, manage named queries, work groups, data catalogs, and query execution history.
